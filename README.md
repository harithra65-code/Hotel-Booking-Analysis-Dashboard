Hotel Bookings Dashboard – Power BI
Project Overview

This Power BI dashboard provides an interactive analysis of hotel booking performance, helping users understand revenue, profit, booking status, customer ratings, discounts, and regional performance. The dashboard is designed to support data-driven decision-making through intuitive visualizations and KPIs.

Dashboard Features :
Treemap – Revenue distribution by hotel category.
Pie Chart – Booking quantity by booking status.
Funnel Chart – Revenue by hotel category.
Gauge Chart – Profit performance.
Line & Clustered Column Chart – Revenue and Profit comparison by category.
Decomposition Tree – Drill-down analysis of revenue by:
-State
-Region
-Category
-Sub-Category
KPI Cards
-Total Discount
-Average Rating
-Unit Price
State-wise Analysis – Performance across different states.

Dataset
The dashboard is built using the Hotel Bookings dataset containing fields such as:
-Category
-Sub Category
-Region
-State
-Revenue
-Profit
-Quantity
-Discount
-Rating
-Unit Price
-Booking Status

Key Metrics
-Total Revenue
-Total Profit
-Booking Quantity
-Average Rating
-Total Discount
-Unit Price

Tools Used
-Microsoft Power BI Desktop
-Power Query
-DAX
-Data Modeling

Insights
-Compare revenue and profit across hotel categories.
-Analyze booking status distribution.
-Identify top-performing regions and states.
-Monitor customer ratings and discounts.
-Drill down into revenue contributors using the decomposition tree.
